# ASC Futevôlei – Scripts

Shell scripts for automating setup, seeding data, and utility tasks.

## Organization

- get_tree_structure.sh: displays project directory tree
- seed_dev_8_users.sh: seeds development environment with 8 users
- seed_dev_users.sh: seeds development environment with users
- set_jwt_token.sh: sets JWT token for authentication
- setup_structure.sh: sets up project structure

## How it Works

- Executed via bash shell
- Used for development setup and data population
- Interact with AWS services and local environment
- Automate repetitive tasks for faster development
